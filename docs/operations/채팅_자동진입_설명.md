# 채팅 시작 방식과 기능 전환에 대해

## 1) 버튼 없이 직접 타이핑해서 시작할 수 있나요?

**네, 가능합니다.**

대화창 하단의 입력창은 퀵스타트 버튼을 누르지 않아도 항상 활성화되어 있습니다. 원하는 내용을 바로 입력하면 대화가 시작됩니다.

단, 이렇게 직접 타이핑해서 시작하면 **일반 AI 모드로 시작**됩니다.

예를 들어 "미팅 준비하자"라고 입력해도, 미팅 준비 전용 기능이 실행되는 게 아니라 일반 AI가 그 내용에 답변하는 방식으로 작동합니다. 미팅 준비, 수당 계산 같은 전용 기능을 쓰려면 해당 퀵스타트 버튼을 눌러 시작해야 합니다.

---

## 2) 대화 중간에 "수당 계산할래"라고 하면 기능이 전환되나요?

**아니요, 전환되지 않습니다.**

현재 구현상, 대화가 시작될 때 어떤 모드(일반 AI / 미팅 준비 / 수당 계산 등)로 시작할지가 정해지고, 그 이후에는 바뀌지 않습니다.

대화 중간에 "수당 계산할래", "미팅 준비해줘" 같은 말을 입력해도 AI는 그 말에 일반적으로 답변할 뿐, 해당 전용 기능으로 전환되지 않습니다.

전용 기능을 사용하려면 **새 대화를 시작하고 해당 버튼을 눌러야** 합니다.

---

## 정리

| 상황 | 결과 |
|------|------|
| 버튼 없이 직접 타이핑 → 대화 시작 | 가능. 단, 일반 AI 모드로 시작 |
| "미팅 준비하자" 타이핑 → 미팅 준비 기능 실행 | 안 됨. 일반 AI가 답변 |
| 대화 중 "수당 계산할래" → 기능 전환 | 안 됨. 기능 전환 없음 |
| 퀵스타트 버튼 → 해당 전용 기능 시작 | 됨. 가장 확실한 방법 |
