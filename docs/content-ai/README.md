# WELLSET Content AI MVP

관심사 기반 콘텐츠를 건강자산 행동으로 자연스럽게 연결하는 주간 추천 기능입니다.

## 관리자 사용 흐름

1. 관리자 메뉴에서 `워크플로우 > Content AI`를 엽니다.
2. 건강, 생활습관, AI·기술, 건강자산, 커뮤니티 비율을 조절합니다.
3. 비율 합계를 100%로 맞춥니다.
4. 균형, 신규 유입, 건강진단 전환, 공유 확산, 브랜드 구축, 시즌 중 목적을 고릅니다.
5. `캘린더 생성`을 누르면 7일 주제·형식·채널·CTA가 추천됩니다.
6. `복사`를 눌러 제작 도구나 협업 문서에 붙여넣습니다.

## 코드 구조

- `src/features/content-ai/types.ts`: 카테고리, 추천 모드, 주제, 캘린더 타입
- `src/features/content-ai/data.ts`: 기본 운영 비율과 검수용 주제 풀
- `src/features/content-ai/recommendation.ts`: 비율 배분, 모드별 점수, 최근 주제 제외, 주간 캘린더 생성
- `src/components/content-ai/content-ai-dashboard.tsx`: 관리자용 전략 설정·추천 화면
- `supabase/migrations/20260718000000_wellset_content_ai.sql`: 전략, 주제, 발행 이력 테이블과 접근 정책

## 추천 원칙

- 카테고리 슬롯은 운영 비율을 7일에 가장 가깝게 환산합니다.
- 최근 발행한 주제 ID는 후보에서 제외할 수 있습니다.
- 모드에 따라 상시성, 시즌성, 공유성, 전환성 점수의 가중치를 달리합니다.
- 확장 유입 콘텐츠는 독립적인 유용성을 먼저 제공하고 관련 건강자산을 연결합니다.
- CTA는 관심(1), 가벼운 참여(2), 서비스 전환(3), 관계 형성(4) 단계로 유지합니다.

## DB 적용

Supabase 마이그레이션을 적용하면 다음 테이블이 추가됩니다.

- `content_strategy_settings`: 캠페인별 운영 비율과 추천 모드
- `content_topics`: 주제 분류, 건강자산 코드, 퍼널, CTA, 품질 점수
- `content_publications`: 추천·예약·발행 이력과 성과 데이터

관리자 외의 로그인 사용자는 승인된 주제만 읽을 수 있고, 등록·수정·삭제는 관리자만 할 수 있습니다.

## 의료 안전

추천 결과는 건강정보와 생활습관 점검을 위한 콘텐츠 초안입니다. 진단·치료·처방을 단정하지 않으며 의료적 판단이 필요한 내용은 전문가 검수를 거쳐 발행합니다.
