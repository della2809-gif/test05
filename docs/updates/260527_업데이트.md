## 수정된 버그

### BUG-C01: 패키지 DB 저장 오류 재발 ✅
- **원인**: `admin_packages` 테이블에 RLS는 활성화되어 있었으나 정책(policy)이 없어 admin 사용자도 쓰기 불가
- **수정**: `20260527000001_admin_packages_rls.sql` 마이그레이션 추가 + Supabase 운영 DB 적용
  - `Anyone can read packages` (SELECT 전체 공개)
  - `Admins can manage packages` (ALL — profiles.role = 'admin'인 사용자)
- **추가**: 오류 시 실제 Supabase error.message가 토스트에 표시되도록 개선

### BUG-C02: 레퍼런스 파일 업로드 재오류 ✅
- **원인**: `after()` 함수 호출 시 런타임 컨텍스트 문제로 업로드 응답 자체가 실패하는 경우
- **수정**: `after()` 호출을 try/catch로 감싸 실패 시에도 업로드 응답은 정상 반환

### BUG-C03: 지플릿 프리뷰 "초기화" 후 네트워크 오류 ✅
- **원인**: 진행 중인 API 요청이 취소되지 않은 상태에서 초기화 → 다음 요청이 간섭받음
- **수정**: `AbortController` 추가
  - `handleSend()`: 이전 요청 abort 후 새 controller로 fetch
  - `handleReset()`: 진행 중인 요청 abort + loading 강제 false로 리셋
  - AbortError는 무시 (토스트 없음), 그 외 오류만 사용자에게 표시

### BUG-C05: 유사IQ 링크 불안정 ✅ (프롬프트 강화)
- 시스템 프롬프트에 `[유사IQ에서 구매하기](링크URL)` 마크다운 형식 명시적 지시

### BUG-C06: 유튜브 자막 추출 오류 ✅
- **원인**: YouTube 페이지 HTML 구조 변경으로 기존 정규식이 실패
- **수정**:
  - User-Agent 헤더 추가 (브라우저 위장)
  - `ytInitialPlayerResponse.captions.playerCaptionsTracklistRenderer.captionTracks` 우선 파싱
  - 브래킷 depth 카운팅으로 중첩 JSON 배열 경계 정확히 추출
  - 언어 우선순위: ko → ko-* 변형 → en → 첫 번째

### BUG-C07: 수당 계산 이미지 인식 불안정 ✅
- **원인**: commission_calc 기능이 텍스트에서만 CVP 숫자를 파싱했음
- **수정**: 이미지 첨부 시 GPT-4o Vision으로 CRM 스크린샷에서 CVP 자동 추출
  - 추출 실패 시 기존 텍스트 파싱으로 graceful degrade

### BUG-C08: 스크립트 DB AI 검색 미작동 ✅ (개선)
- 스크립트 DB 로드 개수 30개 → 50개
- 시스템 프롬프트에 "DB에서 가장 관련성 높은 항목을 그대로 출력, 없는 내용은 만들지 말 것" 규칙 추가

## UX 개선

### IMP-C01: 지플릿 시작 프롬프트 → AI 웰컴 메시지 ✅
- **이전**: initial_prompt가 사용자 발화(초록 말풍선)로 표시됨
- **이후**: 대화 시작 시 AI가 먼저 보내는 흰 말풍선으로 표시

### IMP-C02: 제품 질문 시 DB 원문 그대로 출력 ✅

- 유사IQ 링크 형식 고정: `[유사IQ에서 구매하기](링크URL)`
