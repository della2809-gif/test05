## PDF 파일 업로드 기능 안정화

**왜 만들었나**
관리자 페이지에서 PDF를 업로드할 때 여러 환경적 제약으로 인해 다양한 오류가 발생하던 상황이었음. Vercel 배포 환경의 서버리스 특성, 한글 파일명 처리 한계, 요청 크기 제한 등이 복합적으로 작용하여 파일 업로드 자체가 실패하거나 AI 학습에 반영되지 않는 문제가 있었음. 각 오류의 원인을 개별적으로 파악하여 순차적으로 수정함.

**뭘 할 수 있나**
1. **한글 파일명 업로드** — 파일명에 한글이 포함되어 있어도 오류 없이 정상 업로드됨. 원본 파일명은 그대로 목록에 표시됨
2. **파일 용량 초과 안내** — 4.5MB를 넘는 파일을 올리려 하면, 서버에 전송되기 전에 즉시 용량 초과 안내 메시지가 표시됨
3. **Vercel 환경에서 PDF 내용 추출** — 배포 서버에서 PDF 텍스트 추출이 정상 동작하여 AI 학습(RAG)에 반영됨
4. **대용량 PDF 학습 처리 속도 개선** — 텍스트를 여러 조각으로 나누어 AI 학습용 데이터로 변환할 때 병렬 처리로 전환하여 처리 시간이 약 19초에서 3초대로 단축됨

## 케이스 플로우 응답 스트리밍 전환

**왜 만들었나**
미팅 준비(meeting_prep) 케이스를 시작할 때 AI에게 보내는 지시 내용이 매우 길어, 배포 서버의 응답 제한 시간(10초)을 초과하여 AI 응답을 전혀 받지 못하는 오류가 있었음. 응답이 완성될 때까지 기다렸다가 한 번에 전달하는 방식 대신, 생성되는 즉시 조금씩 전달하는 방식으로 전환하여 해결함.

**뭘 할 수 있나**
1. **미팅 준비 케이스 정상 응답** — 미팅 준비 케이스를 시작해도 타임아웃 없이 AI 응답이 정상적으로 스트리밍되어 표시됨

## AI 응답 URL 클릭 가능하도록 수정

**왜 만들었나**
AI가 응답에서 링크를 `[텍스트](주소)` 형식이 아닌 주소 그대로(`https://...`) 출력하면, 클릭이 불가능한 일반 텍스트로 표시되는 문제가 있었음. 날 주소(bare URL)도 자동으로 클릭 가능한 링크로 변환되도록 수정함.

**뭘 할 수 있나**
1. **날 URL 자동 링크 변환** — AI 응답에 `https://...` 형태의 주소가 그대로 포함되어 있어도 클릭 가능한 링크로 자동 변환됨. 문장 끝 마침표나 쉼표 등 구두점은 링크에서 자동으로 분리됨
