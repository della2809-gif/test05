## 앱·운영자 페이지 명칭 "유사나Q"로 통일

**왜 만들었나**
사용자 화면은 "유사나Q"로 바뀌었으나 운영자(관리자) 페이지와 브라우저 탭, 홈 화면 추가 시 이름 등 곳곳에 옛 이름(GENIEA)이 그대로 남아 있었음. 명칭이 화면마다 달라 혼란을 줄 수 있었음.

**뭘 할 수 있나**
1. **명칭 일관성** — 운영자 페이지 설명, 브라우저 탭 제목, 체험 만료 안내, 홈 화면 추가 이름 등 옛 이름이 남아있던 곳이 모두 "유사나Q"로 통일됨

## 케이스 시작 멘트 숨김 (AI 가이드는 유지)

**왜 만들었나**
케이스를 시작하면 사용자가 직접 입력한 것처럼 "케이스를 시작해주세요"라는 메시지가 화면에 떠서 어색했음. 사용자 입장에서는 이 멘트가 보이지 않는 것이 자연스러움.

**뭘 할 수 있나**
1. **시작 멘트 미노출** — 케이스를 시작해도 "케이스를 시작해주세요" 문구가 보이지 않음
2. **가이드 자동 표시 유지** — 케이스 진입 시 AI 안내(단계 목차)는 그대로 바로 표시됨

## 자동견적 카드에 제품 링크 표시

**왜 만들었나**
자동견적 결과에서 어떤 제품은 링크가 나오고 어떤 제품은 안 나와 일관성이 없었음. 견적 카드 자체에는 제품 링크가 빠져 있었음.

**뭘 할 수 있나**
1. **제품명 클릭 이동** — 자동견적 카드의 제품명을 누르면 해당 제품의 유사나Q 상세 페이지로 이동함 (제품 정보에 링크가 입력된 제품에 한함)

## 케이스 단계의 유튜브·이미지 링크 일관 연결

**왜 만들었나**
어떤 케이스(키리더 가이드)는 단계에서 실제 유튜브 강의 링크가 잘 나왔는데, 똑같이 만든 다른 케이스(5 STEP)는 링크 대신 "youtube", "image" 같은 내부 단어만 나왔음. 케이스마다 결과가 들쭉날쭉했음. 원인은 내부 식별자가 AI에게 그대로 전달되어 AI가 그 단어를 따라 출력한 것이었음.

**뭘 할 수 있나**
1. **실제 링크·이미지 제공** — 어느 케이스에서든 단계에 연결된 자료가 있으면 "youtube"·"image" 같은 글자 대신 실제 클릭 가능한 유튜브 강의 링크와 이미지가 표시됨
2. **케이스 간 일관성** — 모든 케이스(5 STEP·키리더·미팅 준비·신규 제품 등)에 동일하게 적용됨

## PC 이미지 업로드 용량 초과 안내

**왜 만들었나**
PC에서 큰 사진을 올리면 "원인 모를 실패"로만 보여 사용자가 이유를 알 수 없었음. 서버가 허용하는 용량(약 4.5MB)을 넘으면 안내 없이 실패하는 구조였음.

**뭘 할 수 있나**
1. **용량 초과 안내** — 4.5MB를 넘는 파일을 올리면 "용량이 커서 업로드할 수 없으니 줄여서 올려달라"는 안내가 명확히 표시됨 (업로드 전과 서버 거부 시 모두 안내)

> 참고: 큰 파일을 자동으로 줄여 올려주는 기능은 아니며, 사용자가 직접 용량을 줄여야 함. 작은 사진(4.5MB 이하)인데도 PC에서 안 되는 경우는 용량 외 다른 원인이므로 추가 확인이 필요함.
