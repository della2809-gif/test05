# 2026-07-18 업데이트

## WELLSET Content AI MVP

### 추가 기능

- 관리자 메뉴의 `워크플로우`에 `Content AI` 화면을 추가했습니다.
- 5개 콘텐츠 카테고리 운영 비율을 5% 단위로 조정할 수 있습니다.
- 균형, 신규 유입, 건강진단 전환, 공유 확산, 브랜드 구축, 시즌 추천 모드를 추가했습니다.
- 18개 기본 주제에서 7일 콘텐츠 캘린더를 생성합니다.
- 추천 결과에 주제, 유입 관심사, 연결 건강자산, 형식, 채널, CTA 단계, 추천 이유를 표시합니다.
- 캘린더 전체를 텍스트로 복사할 수 있습니다.

### 추천 로직

- 운영 비율을 주간 7개 슬롯에 최대한 가깝게 배분합니다.
- 최근 발행 주제 제외 기능을 엔진 수준에서 지원합니다.
- 모드별로 상시성, 시즌성, 공유성, 전환성 가중치를 다르게 적용합니다.
- 신규 유입 모드에서 AI·기술과 라이프스타일 주제를 우대합니다.

### 데이터베이스

- `content_strategy_settings` 테이블: 캠페인 운영 비율과 추천 모드
- `content_topics` 테이블: 콘텐츠 주제의 4단계 분류, CTA, 품질 점수
- `content_publications` 테이블: 예약·발행 이력과 성과
- 관리자 전체 권한과 승인 주제 읽기 정책을 추가했습니다.
