## 자유 채팅 제품 정보 정확도 (CR-037)

**왜 만들었나**
자유 채팅에서 "헬스팩 알려줘"처럼 제품 이름만 물으면, 제품 DB 기반의 정확한 정보·가격·상세 링크가 아니라 일반 종합영양제 설명처럼 나오는 문제가 있었음. 제품 키워드("제품·영양제·가격" 등)가 함께 있어야만 DB를 참고하도록 막혀 있었던 것이 원인임. 제품명·별칭만으로도 정확히 답해야 한다는 요청을 반영함.

**뭘 할 수 있나**
1. **제품명/별칭 질의 정확 응답** — 자유 채팅에서 "헬스팩 알려줘"처럼 제품 이름·별칭만 입력해도 제품명·가격·유사IQ 상세 링크가 출력됨.
2. **잡담 오작동 방지** — 제품과 무관한 대화에는 엉뚱한 제품 목록이 끼어들지 않음. 실제로 관련 있을 때만 제품 정보가 따라옴.

## 검색 정확도 — 조사 처리 (CR-008)

**왜 만들었나**
검색이 단어에 붙은 한국어 조사 때문에 빗나가는 문제가 있었음. "헬스팩을", "보상플랜을"처럼 조사가 붙으면 DB의 "헬스팩", "보상플랜"을 못 찾았음. 제품·스크립트·미팅·유튜브 검색 전반에 영향이 있었음.

**뭘 할 수 있나**
1. **조사 붙은 검색어도 적중** — "헬스팩을", "비타민이", "보상플랜을"처럼 조사가 붙어도 원래 단어를 찾아냄.
2. **멀쩡한 단어 보호** — "오메가"처럼 끝글자가 조사와 같은 정상 단어는 깨지지 않도록 안전장치를 둠.

## 지플릿 아이콘 모노톤 정리 (CR-002)

**왜 만들었나**
지플릿 메뉴 아이콘이 기능마다 색이 달라 "형형색색"으로 보였음. 클라이언트가 원한 것은 깔끔한 모노톤(단색 통일) 느낌이었음. 알록달록한 이모지를 선형 아이콘으로 바꾼 뒤에도 색이 제각각이라 재정리가 필요했음.

**뭘 할 수 있나**
1. **단색 통일** — 모든 지플릿 아이콘 색을 하나의 중립색으로 통일해 깔끔한 모노톤으로 보임.
2. **기능 구분 유지** — 색이 아니라 아이콘 모양으로 기능을 구분하므로 어떤 메뉴인지 알아보기 쉬움.

## 채팅 이미지 확대뷰 개선

**왜 만들었나**
채팅에 표시되는 이미지가 잘려 보이거나(목록에서 비율이 깨짐) 확대뷰가 작고 답답한 문제가 있었음. 이미지 자료를 또렷하게 확인할 수 있어야 함.

**뭘 할 수 있나**
1. **비율 유지 표시** — 채팅 속 이미지가 잘리지 않고 원래 비율로 보임.
2. **큰 확대뷰** — 이미지를 누르면 화면 대부분을 채우는 큰 확대뷰로 또렷하게 볼 수 있음.

## 패키지 구성품 보존 (CR-011)

**왜 만들었나**
관리자 화면에서 패키지를 수정·저장하면 기존 구성품 정보가 빈 값으로 덮여 사라지는 버그가 있었음. 구성품 편집 화면이 따로 없는 상태에서 다른 항목만 고쳐도 구성품이 날아갔음.

**뭘 할 수 있나**
1. **구성품 보존** — 패키지 이름·가격 등만 수정해도 기존 구성품이 그대로 유지됨.
2. **신규 생성 정상** — 새 패키지를 만들 때는 빈 구성품으로 정상 시작함.

## 레퍼런스 DB 활성/비활성 토글 (CR-034)

**왜 만들었나**
레퍼런스 자료(실전미팅 원본, 책 PDF 등)를 보관은 하되, 필요할 때 챗봇 답변·검색에서 제외할 수 있어야 함. 기존에는 켜고 끄는 기능 자체가 없어 한번 넣은 자료가 항상 검색에 노출됐음. 이미지 DB·책 PDF·실전미팅 자료를 본격 입력하기 전에 선행되어야 하는 기능임.

**뭘 할 수 있나**
1. **자료별 켜기/끄기** — 관리자 레퍼런스 화면에서 파일마다 전원 버튼으로 활성/비활성을 전환함. 비활성 자료는 흐리게 표시되고 "비활성" 표식이 붙음.
2. **보관 유지** — 비활성 상태여도 파일은 삭제되지 않고 그대로 보관되며, 챗봇 답변에서만 제외됨.

> 배포 상태: DB 구조 변경(마이그레이션)과 코드 배포까지 완료됨. 로그인 후 관리자 화면에서 파일별 토글 동작과 챗봇 검색 제외 여부만 수동 확인하면 됨.
