# 점수·등급·우선순위

실행 원본: `src/features/health-check/scoring.ts`

## 영역 상태

| 영역점수 | 코드 | 표시문구 | 기본 안내 |
|---:|---|---|---|
| 0~3 | good | 양호 | 현재 건강자산을 잘 유지하고 있습니다. |
| 4~6 | attention | 보통 | 일부 생활습관 개선이 필요합니다. |
| 7~10 | warning | 경계 | 원인 확인과 적극적인 관리가 필요합니다. |
| 11~15 | risk | 위험 | 전문상담과 체계적인 관리가 권장됩니다. |

## 건강 종합점수

```text
건강점수 = round(100 - (총 위험점수 / 180) × 100)
```

| 건강점수 | 등급 |
|---:|---|
| 85~100 | 건강자산 우수 |
| 70~84 | 대체로 양호 |
| 55~69 | 관리 필요 |
| 40~54 | 적극 관리 필요 |
| 0~39 | 집중 관리 권장 |
