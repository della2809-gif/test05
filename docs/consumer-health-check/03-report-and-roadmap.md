# 결과 리포트와 4주 로드맵

실행 원본:

- `src/features/health-check/scoring.ts`
- `src/components/health-check/health-result.tsx`

## 리포트 7개 섹션

1. 건강 종합평가
2. 건강 상태 요약
3. 주목해야 할 영역 TOP 3
4. 원인 후보 분석
5. 맞춤 건강관리 프로그램
6. 4주 건강관리 로드맵
7. 건강 코치의 제안

## 4주 로드맵 기본 구조

### 1주차: 첫 번째 우선영역 기반 만들기

- 기상·식사 시간 기록
- 가장 쉬운 행동 한 가지를 매일 반복

### 2주차: 두 번째 우선영역 균형 잡기

- 식사 후 10분 가볍게 걷기
- 수분 섭취와 컨디션 함께 기록

### 3주차: 운동·순환·근육

- 주 3회 20분 걷기
- 주 2회 가벼운 근력·균형 운동

### 4주차: 습관 정착과 재평가

- 주간 기록 돌아보기
- 유지할 습관 2가지 선택

## 문구 작성 원칙

- 확정적인 효과를 약속하지 않습니다.
- 질병을 진단하지 않습니다.
- 입력되지 않은 질환을 추측하지 않습니다.
- 제품 판매 문구처럼 작성하지 않습니다.
- `가능성`, `관련될 수 있음`, `기대할 수 있음`처럼 보수적으로 표현합니다.
