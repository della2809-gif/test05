# GENIEA Docs

이 폴더는 앱 코드가 아닌 프로젝트 운영 문서의 단일 루트다.

## 폴더 기준

- `updates/`: 날짜별 개발/운영 업데이트 공유 문서.
- `operations/`: 운영정책, DB 반영 기록, QA, 조사 결과, 클라이언트 요청 정리.
- `handovers/`: 다음 작업자 인수인계와 핸드오프 문서.
- `prompts/`: 지플릿, 시스템 프롬프트, 프롬프트 반영 대기 문서.
- `training/`: 강의안, 슬라이드, 교육 자료.
- `archive/`: 과거 도구 산출물, 마이그레이션된 위키, 보존 목적 문서.

## 루트 배치 원칙

새 문서는 기본적으로 위 하위 폴더 중 하나에 둔다. `docs/` 루트에는 이 README처럼 폴더 전체를 안내하는 문서만 둔다.
